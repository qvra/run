# qvra hello

Fast contact artifact for `run`.

This is the smallest real utility surface in Class A.
It proves that `run` is no longer only declared — it is executable.

## What it does

Prints a clean `qvra` contact message with timestamp and working directory.

## Run

```bash
bash hello.sh
````

## Output

* qvra marker
* UTC timestamp
* current directory
* execution confirmation

## Why it exists

This is not a toy placeholder.
It is the first live executable contact route inside `run`.
It establishes the exact standard for future small tools:

* one file
* one command
* visible output
* no friction
